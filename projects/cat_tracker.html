Cat Tracker

Real-time cat detection & tracking with remote laser control

Overview

Cat Tracker runs a lightweight vision model to find and track a cat in view. A small pan/tilt laser module can “play” within predefined safe zones via a web UI, all on top of the Shadow Flask video stream.

Build Process

  1. Model & Pipeline: Choose a fast model (e.g., YOLO-nano/YOLOv8n); resize and run inference on a Pi or small GPU device.
  2. Tracking: Centroid/Kalman tracker for smooth motion and short occlusion handling.
  3. Zones & Safety: Define valid “play area,” clamp angles, and add an emergency stop.
  4. Laser Control: PWM control for servos/galvos; Flask endpoints to command positions.
  5. Web UI: Stream preview with “chase” toggle and manual joystick control.
  6. Metrics: FPS, detection confidence, and laser position logging.

Results

  • Reliable cat detections with smooth retargeting at interactive frame rates.
  • Remote “playtime” from phone or laptop via a simple web interface.
  • Safety zones prevent shining in restricted directions or at sensitive areas.
