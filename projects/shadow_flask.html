Shadow Flask

Raspberry Pi + USB camera streaming over a private Tailscale network

Overview

A minimal Flask app on Raspberry Pi that streams USB-camera video over a secure, peer-to-peer Tailscale network—viewable from any authorized device’s browser.

Build Process

  1. Pi Setup: OS image, camera sanity checks, virtualenv for Python deps.
  2. Flask App: MJPEG endpoint; adjustable resolution/FPS; basic auth.
  3. Serviceization: Systemd unit for auto-start and restart on failure.
  4. Tailscale: Join private mesh; ACLs; device tags for access control.
  5. Mobile UX: Simple HTML page with inline video and latency notes.
  6. Logging: Rotating file logs for uptime/health checks.

Results

  • Reliable sub-second preview in local networks; secure remote access via Tailscale.
  • Low resource usage; boots straight into a ready stream.
  • Easy extension point for capture, screenshots, and alerts.
